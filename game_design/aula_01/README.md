# O que faz um game designer?

- Jogar é importante para um GD, mas `foi pra escola de game design` => `não tem tempo pra jogar`
- Jogar jogos variados: `esporte`, `corrida`, `gatcha`, etc.
  - Referência e repertório é importante para a arte.
- Saber sobre arte, mas não é artista: não é animador, não é desenhista.

## EXPERIÊNCIA

O GD é responsável pela **EXPERIÊNCIA**.

- medo
- frenesi
- frustração
- superação

O GD é a base, a primeira atividade:

- o mood-board precisa evocar essa emoção
- o roteiro deve ...

### Ciclo

1. `vender ideias`: para stakeholders - programador, ilustrador, líder, player, etc.

- cada audiência um discurso específico
- GDD
- **OUVIR**

2. `planejar`: ajuda o produtor a priorizar

3. `executar`

- narrativa: qual o contexto do jogo
- system: como as coisas funcionam e interagem
- level: como serão os ambientes
- economia / balanceamento: números estão equilibrados

**TUDO** no objetivo da experiência
